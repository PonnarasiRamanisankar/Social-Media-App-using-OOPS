## Overview

This is a console-based Social Media Application built in Java using Object-Oriented Programming principles. The application allows users to sign up, connect with friends, send and manage friend requests, create posts, view a news feed from friends, and interact with posts. The project demonstrates modular design using a DAO-Service-Main layered architecture and Java packages.

## Project Structure

The project is organized into the following structure:

* model: Contains entity classes, including User and Post
* dao: Handles data storage and basic operations for users, friends, and posts
* service: Contains business logic for managing users, friends, and posts
* Main.java: Runs the console application and provides a menu-driven interface

## Features

1. User Management

   * Sign up with Name, Date of Birth, Age, Location, and Occupation
   * View all users

2. Friend Management

   * Send friend requests
   * Accept or reject friend requests
   * View friend list
   * Suggest friends

3. Post Management

   * Create posts visible to friends
   * News feed displays posts from friends
   * Posts show likes and comment counts

4. Notifications

   * Receive notifications for friend requests
   * Can be extended to include post interactions

## How to Use

1. Open the application.
2. Sign up new users with personal details.
3. Send friend requests to other users and manage received requests.
4. Create posts and view the news feed to see friends’ posts.
5. Interact with posts by liking and commenting (functionality can be extended).

## Example Scenario

* Alice, Bob, and Charlie sign up.
* Alice sends a friend request to Bob.
* Bob accepts the request.
* Alice creates a post: “Enjoying the sunny day in New York! #blessed”
* Bob sees Alice’s post in his news feed.

## Technologies Used

* Java SE
* Object-Oriented Programming concepts
* Console-based application
* Packages for modular design
